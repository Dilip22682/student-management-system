🎓 Student Management System
A full-featured Student Management System built using Django (CBV + FBV concepts) that allows administrators to manage student records efficiently with CRUD operations and clean UI.

🚀 Live Demo

🌐 https://student-management-system-data.onrender.com

📌 Project Overview

This project demonstrates:

Django Models & ORM

ModelForms

Class-Based Views

URL Routing

Bootstrap UI Integration

Success Messages Framework

Git Version Control

Deployment on Render

✨ Features
Feature	Description
➕ Create	Add new student records
📋 Read	View student list
✏ Update	Edit existing student details
❌ Delete	Remove student records
🔎 Validation	Form validation with error handling
🎨 UI	Responsive Bootstrap design
📦 Deployment	Hosted on Render
🛠️ Tech Stack

Backend: Python, Django

Frontend: HTML, CSS, Bootstrap 5

Database: SQLite (Development)

Deployment: Render

Version Control: Git & GitHub
